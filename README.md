# Project Title

Fake News articles

## Getting Started

### Installing

* Clone or Fork

* Init local MAMP, WAMP, LAMP or XAMPP instance.

* SQLite3 needs to be installed for this to work.
	If you're on Mac or Linux run
	```
	sudo apt-get install sqlite3
	```
* Visit localhost:80/index.php
